For Running Genymotion, we need these rpms
1.virtualbox - download from oracle
2.genymotion - download ubuntu version lastest bin file (no fedora version is added but we can use)
3.dependencies - download require dependencies from opensuse site (i don't know where can i find for fedora) https://mirrors.fedoraproject.org/mirrorlist?repo=fedora-23&arch=x86_64
4.installation

## Installation ##
1.Virtualbox
2.Genymotion
3.do this vvvv

sudo chmod +x genymotion-2.6.0-ubuntu15_x64.bin <<<-- make the bin executable
sudo ./genymotion-2.6.0-ubuntu15_x64.bin 	<<<-- install for every user
cd opt/genymobile/genymotion/		 	<<<-- will install to opt
./genymotion                             	<<<-- you will see require libraries
